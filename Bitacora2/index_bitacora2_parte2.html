Root

El selector: root coincide con el elemento raíz del documento. Hace referencia al elemento raíz (al documento HTML).
En la mayoría d ellos casos, :root se refiere al elemento <html> en una página web. Independientemente del lenguaje de marcado, :root siempre seleccionara al elemento superior del documento.
La speudo clase :root puede ser útil para declarar variables CSS globales.

Propiedades personalizadas

Variables CSS
En CSS, las propiedades personalizadas (también conocidas como variables) son entidades que contienen valores específicos que se pueden volver a utilizar en un documento.
Los sitios web complejos tienen cantidades muy grandes de CSS, a menudo con una gran cantidad de valores repetidos.
Además, las propiedades personalizadas pueden ser animadas, lo que brinda flexibilidad adicional en la creación de efectos dinámicos en las interfaces web.

Variables CSS

flex-direction
Las varables cSS permiten que un valor se almacene en un lugar y luego se haga referencia en varios otros lugares.

Se declara con la siguiente sintaxis:
--main-color: red; -- (nombre de la variable) : (valor de la variable)
Y se invoca:
background-color: var (--main color);

Propiedades personalizadas

Para declarar propiedades personalizadas se debe usar un nombre que comience con dos guiones (--), y un valor que puede ser cualquier valor valido en CSS.
Es importante el uso de identificadores semánticos.
El alcance (ámbito) en el que podremos usar la variable, dependerá del selector donde se declare la variable.
Las propiedades personalizadas heredan.
