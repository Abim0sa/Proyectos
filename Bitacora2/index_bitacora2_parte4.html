CSS Reset y Normalize

CSS Reset: Elimina el estilo predeterminado del navegador para proporcionar una base consistente en todos los navegadores.

Normalize: Normaliza los estilos predeterminados de los elementos HTML en todos los navegadores para una experiencia más coherente.

Enfoque y alcance

Reset Aplica estilos mínimos a todos los elementos HTML, reiniciando por completo las reglas de estilo.

Normalize Establece estilos base consistentes para los elementos HTML, manteniendo ciertos estilos predeterminados que se consideran útiles.

Uso recomendado

Reset Adecuado para proyectos donde se requiere un control total sobre el diseño y se está dispuesto a reconstruir estilos desde cero.

Normalize Ideal para proyectos donde se prioriza la consistencia entre navegadores y se prefiere una base más estable para comenzar a trabajar.

Complementariedad
Ambos enfoques pueden combinarse según las necesidades del proyecto. Por ejemplo, puedes utilizar Normalize para establecer una base coherente y luego aplicar un CSS Reset para personalizar los estilos según tus preferencias específicas.

Reset

margin, padding, y border:
Se establecen a 0 para todos los elementos, lo que elimina los espacios predeterminados entre elementos.

font-size, line-height, font-family:
A menudo se restablecen para garantizar que no haya estilos heredados.

list-style:
Elimina los estilos de viñetas o numeración predeterminados de las listas.
Requiere más trabajo para reconstruir estilos desde cero, pero ofrece un control total sobre el diseño.

Normalize

box-sizing:
Establece el modelo de caja a border-box, lo que ayuda a evitar problemas de diseño relacionados con el cálculo del tamaño del elemento.

font-weight, font-style:
Normaliza los estilos de fuente para garantizar la consistencia entre navegadores.

line-height:
Ajusta el espaciado entre líneas para que sea más consistente entre diferentes elementos de texto.
