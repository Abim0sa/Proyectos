Tipografía responsive

El texto es parte imprescindible de la web. Lo que se quiere conseguir se muestra mediante el texto. Lo más importante es que el texto se pueda leer. Pensar en la legibilidad es un factor clave en los beneficios que puedes generar con tu sitio web.
Junto con la tipografía, se puede alterar por completo el significado que asociamos con un proyecto.
Añade diversidad estética, lo cual contribuye a una experiencia de usuario más atractiva y satisfactoria.

Legibilidad

El principal objetivo es la facilidad con la que se puede leer y comprender un texto.
El tamaño de la letra
Si el texto es menor a 16px, puedes estar cometiendo un error. 16 pixeles es el fontsize que viene determinado en la mayoría de los navegadores. Se debe crear un ambiente a agradable para el usuario.
Número de palabras por línea
el acho de línea, es otro factor decisivo. El ancho de línea optimo contiene entre35 y 65 caracteres (7-12 palabras aproximadamente).
Debemos definir un ancho para la caja de texto y contar cuantos caracteres entran en función del tamaño.

Combinación tipográfica

Usar diferentes familias tipográficas sirve ... Menos, es más.
Ofrece jerarquia visual para el contenido, destacando elementos importantes como titulares y llamadas a la acción.
Interlineado
En la web el interlineado lo podems ajustar con la propiedad alto de línea o line-height, que es la distancia entre dos líneas.
La recomendación es establecer el line-height al 120% (multiplicar por 1.2)

Color y unidades rem

El color
Uno de los principales errores que se cometen es emplear combinaciones de colores innecesarias.
El w3C sugiere que se use el contraste adecuando o se podría terminar abrumando al usuario.

Unidades rem
Significa "unidad de tamaño de fuente raíz" (root em), y se refiere al tamaño de fuente del elemento raíz del documento HTML.
