Modulo flexbox

El módulo flexbox tiene como objetivo proporcionar una manera más eficiente de diseñar, alinear y distribuir el espacio entre los elementos en un contenedor, incluso cuando su tamaño es desconocido y/o dinámico.
Otorga al contenedor la capacidad de modificar al ancho, alto y el orden de sus elementos para llenar mejor el espacio disponible.
Un contenedor flexible expande los elementos para llenar el espacio libre disponible o los reduce para evitar el desbordamiento. El módulo flexbox hace que los elementos dentro de un contenedor trabajen en un solo eje.

Contenedor flexible y elementos secundarios

Flexbox en HTML y CSS ofrece un enfoque poderoso para el diseño web, permitiendo la creación de diseños responsivos y dinámicos. Al utilizar propiedades como display: flex, flex-direction, flex-wrap, justify-content, align-items y align-content en los contenedores flexibles, junto con flex y align-self en los elementos hijos, se puede controlar eficazmente la disposición y el alineamiento de los elementos en una página web.

Propiedades para el contenedor flexible

align-items
Determina cómo se alinean los elementos verticalmente dentro del contenedor.

align-content
Esta propiedad alinea los renglones de un contenedor flexible cuando hay espacio extra en el eje transversal.

Propiedades para los hijos

flex-basis: auto;
La longitud es igual a la longitud del elemento flexible.

flex-basis: initial;
Establece esta propiedad a su valor predeterminado.

flex-basis: numero;
Unidad de longitud, o porcentaje, que especifica la longitud inicial de los elementos flexibles.

Flex-inherit
Hereda esta propiedad de su elemento padre.

Contenedor flexible

flex-direction
Esto establece el eje principal definiendo la dirección en que se colocan los elementos.

flex-wrap
Por defecto todos los elementos intentaran encajar en una sola línea.

flex-flow
Definen los ejes principal y transversal del contenedro flexible. El valor determinado es la row nowrap.

flex-flow
Definen los ejes principal y transversal del contenedro flexible. El valor determinado es la row nowrap.

justify-content
Esto define la alineación a lo largo del eje principal.

Gap
Esta propiedad controla el espacio entre elementos flexibles Se aplica ese espacio solo entre elementos que no están en los bordes exteriores.

Propiedades para los hijos

order
Controla el orden en el que aparecen en el contenedor flexible.

flex-grow
Define la capacidad para que un ítem crezca si es necesario. Acepta un valor entero que sirve como proporción. Indica que cantidad de espacio disponible disponible dentro del contenedor flexible debe ocupar el ítem.

flex-shrink
Especifica la capacidad de un elemento para reducir en relación con el resto de los elementos flexibles del contenedor. Por default es 1.

flex-basis
Esto define el tamaño predeterminado de un elemento antes de distribuir el espacio restante.

align-self
Modifica, alinea solo al elemento especificado.
